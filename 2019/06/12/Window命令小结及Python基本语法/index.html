title: Window命令小结及Python基本语法
date: 2019-06-12 19:17:46
tags:
— ###一：windows常用命令###
cd +路径 进入该位置
cd.. 在原路径的基础上，退一层
md +路径+文件夹名 在此路径下建立文件夹
cls 清屏
chdir 查看当前路径
ls -l 查看当前目录下文件
.. -v 查看版本

关于hexo的命令

cnpm install -g hexo-cli 安装博客框架
hexo init 初始化博客
hexo s 启动hexo
hexo n+ “文章名字” 新建博客文章
vim+ 博客文件 打开vim编辑器，编写博客
hexo clean 清理博客（不懂啥意思）
hexo g 生成博客（在本地）
hexo d 将博客部署到远端
— ###二：Python基本语法###
1.数组

2.列表

3.字典
dict={‘one’:1,’two’:2}
以键值对的形式储存
有四种输出方式
dict
通过键
dict.keys()
dict.values()
4.requests的使用
requests.get()
r.starus_code
